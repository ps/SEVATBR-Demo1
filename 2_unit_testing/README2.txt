# Unit Tests #
The unit tests are in several bash scripts in this directory.

To run them, run them in bash. E.g.

./computer-vision-tennis-ball.sh

Will run the unit tests regarding image recognition of the tennis ball.

## Dependencies ##
There are several dependencies required to run these unit tests. They include:

- Linux
- Python 2.7
- SimpleCV Python library
- OpenCV v2
- Google App Engine
- Flask
- Gstreamer 0.10
- SphinxBase
- Pocketsphinx
- Festival

** This is not the full list of deps for demo 2, only for demo 1. **
